# OnlyCards Web Monorepo

## Модули

### Product

Содержит типы продуктовых сущностей и общую логику работы с ними.

### UI

Базовые стили, компоненты карточек и колоды.

Импортирует Product,
чтобы карточка могла принимать атрибутом продуктовую сущность.

### Service

Предоставляет функции для работы с удалёнными данными.

Импортирует Product, чтобы функции возвращали продуктовые сущности.

### Landing

Главная страница приложения. Содержит описание и презентацию.

Импортирует UI и Product,
чтобы включать в презентацию демонстрацию работы с карточками.

### Admin

Приложение для преподавателей.

Импортирует Product, UI и Service.

### Client

Приложение для учеников.

Импортирует Product, UI и Service.

## Команды

### Установка зависимостей

```
npm i
```

Помимо внешних зависимостей, модули опираются на другие модули.
Чтобы не было проблем, лучше сразу собрать всё.

### Сборка

Собрать всё:

```
npm run build
```

Эта команда собирает модули в правильном порядке, исключая проблемы.

Собрать отдельные модули:

- `npm run build:product`
- `npm run build:ui`
- `npm run build:service`
- `npm run build:client`
- `npm run build:admin`
- `npm run build:landing`

Некоторые модули имеют в зависимостях другие модули.
Например, если попытаться собрать `client` до сборки `ui`, будет ошибка.

### Проверка качества кода

#### Соответствие типов

```
npm run ts
```

#### Стиль кода

```
npm run lint
```

#### Автоматическое тестирование

```
npm run test
```
