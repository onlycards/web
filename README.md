# OnlyCards Frontend Monorepo

OnlyCards помогает преподавателям иностранных языков выдавать
ученикам наборы слов, а ученикам — запоминать эти слова по карточкам.

В этом репозитории только фронтенд: приложение, админка и лендинг.
Технологии: TypeScript, React, Zustand, SCSS, Vite.

## Модули

### Service

Предоставляет типы и функции для работы с удалёнными данными.

### UI

Стили и компоненты карточек и колоды.

Импортирует Service,
чтобы карточка могла принимать атрибутом продуктовую сущность.

### Landing

Витрина приложения. Содержит описание и презентацию.

Импортирует Service и UI,
чтобы включать в презентацию демонстрацию работы с карточками.

### Admin

Приложение для преподавателей.

Импортирует Service и UI.

### Client

Приложение для учеников.

Импортирует Service и UI.

## Команды

### Установка зависимостей

```
npm i
```

Помимо внешних зависимостей, модули опираются на другие модули.
Чтобы не было проблем, лучше сразу собрать всё.

### Сборка

Собрать всё:

```
npm run build
```

Эта команда собирает модули в правильном порядке, исключая проблемы.

Собрать отдельные модули:

- `npm run build:ui`
- `npm run build:service`
- `npm run build:client`
- `npm run build:admin`
- `npm run build:landing`

Некоторые модули имеют в зависимостях другие модули.
Например, если попытаться собрать `client` до сборки `ui`, будет ошибка.

### Проверка качества кода

#### Соответствие типов

```
npm run ts
```

#### Стиль кода

```
npm run lint
```

#### Автоматическое тестирование

```
npm run test
```
